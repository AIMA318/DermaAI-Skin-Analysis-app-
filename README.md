🌸 DermAI - AI-Based Personalized Skincare Routine Generator

📌 Project Overview

DermAI is an AI-powered skincare recommendation system that generates personalized skincare routines based on a user's skin type, skin concerns, age, and environmental factors.

The system analyzes user inputs and provides customized skincare recommendations including cleansers, serums, moisturizers, sunscreens, and treatment products. It helps users follow an effective skincare routine without requiring professional consultation.

---

🎯 Objectives

- Identify user skin type and concerns.
- Generate personalized AM and PM skincare routines.
- Recommend suitable skincare ingredients.
- Predict hydration levels using machine learning.
- Provide easy-to-understand skincare guidance.

---

✨ Features

- Personalized skincare recommendations
- Morning and Night routine generation
- Skin concern analysis
- Ingredient recommendations
- Machine Learning based predictions
- User-friendly interface
- Dataset-driven recommendations

---

🧠 Machine Learning Models

Classification Model

Used to predict skin type and skincare category.

Algorithms:

- Decision Tree Classifier
- Random Forest (Future Enhancement)

Regression Model

Used to predict hydration score.

Algorithms:

- Linear Regression
- XGBoost Regressor (Future Enhancement)

---

⚙️ System Workflow

1. User enters skincare information.
2. Data is preprocessed and encoded.
3. Classification model predicts skin type.
4. Regression model predicts hydration score.
5. Recommendation engine generates skincare routine.
6. Personalized results are displayed.
